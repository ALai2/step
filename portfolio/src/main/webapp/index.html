Interesting Projects

Besides the Unity project that I mentioned above, another project that I did that I find particularly interesting is Zing. Zing is a project that I started with two people over the summer my freshman year when I did an internship at Life Changing Labs in Ithaca. Life Changing Labs is an entrepreneurship program which matches founders with interns that they work with to build their startup idea. Zing is a project that brings isolated students on campus together and during this time of quarantine we actually made a variant of Zing called Quarantine Buddy which pairs users to do a Zoom call together so they don't have to feel as isolated when they are at home. A special shoutout to my teammate Sam who got the opportunity to be on ABC news with Kelly and Ryan to talk about Quarantine Buddy.

I also worked on a bomb defusal game with the C programming language on a single FRDM MK64F board with my friend. We used the board's built-in accelerometer to determine orientation values for displaying the different sections of the game. Each section had a different mini-game to play: Morse code translation, simon says, reaction time, and a last stand game which depended on the results of the other 3 mini-games. The game uses the LED to display a randomly selected sequence of colors based on the rules of each mini-game and the user uses the pushbuttons to play the game.
